# Performance_Review_and_Evaluation

**Purpose:** Detects performance management and review materials.
**Detection Logic:** ≥2 performance terms; or 1 performance term + Employee/Manager context.
**Typical Examples:** Performance forms, review summaries, feedback notes.

**Confidence Mapping (suggested):**
- High ≈ 85% — strict pattern / ≥2 primary + context term within proximity
- Medium ≈ 65% — ≥2 primary terms or looser pattern
- Low ≈ 40% — single primary term (title/header/body)
